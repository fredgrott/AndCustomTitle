Slide #13

  • Using only default settings in application tag in AndroidManifest still gives a title bar flash, why?

Slide #14

  • Using Default AndroidManifest settings applies the title bar default style theme before onCreate fully initializes, hence the title bar flash even though you have called

    requestWindowFeature(Window.FEATURE_NO_TITLE);

Slide #15

So we setup a splash activity that posts a delayed intent request on a thread and we set the androidManifest application tag to use this theme

1 <application android:icon="@drawable/andcustomtitleicon"
2   android:label="@string/app_name"
3     android:theme="@android:style/Theme.NoTitleBar">
